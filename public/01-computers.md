## Свойства компьютера к которым мы привыкли

### Hardware - физическая неизменяемая реализация

Так называемая *архитектура*, например - x86, ARM, RISC-V

### Software - программное обеспечение

Может быть произвольно сложным - от десятков инструкций для гирлянд до миллиардов инструкций для операционной системы

### RAM - Random Access Memory

Необязательный элемент. Свойства мы определим позднее

### IO (Input / Output) - Устройства взаимодействия с компьютером

Мышь, клавиатура, монитор

Накопители данных (SSD, Жесткие диски, флешки)

Power LED

Сетевая карта

### Принципы

#### Адресность

- RAM - память произвольного доступа
- Ячейки фиксированного размера
- Фиксированные номера ячеек (адрес)

#### Однородность памяти

- И код и данные в одной памяти
- По значению или номеру ячейки нельзя определить что в ней хранится
- Смысл значения в ячейке определяется динамически

#### Программное управление

- Кодируется инструкциями процессора
- Инструкции хранятся в оперативной памяти
- Инструкции исполняются последовательно
- Специальные инструкции ветвления
